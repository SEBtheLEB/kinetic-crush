# Play Store Asset Checklist

- App icon: 512x512 PNG
- Adaptive icon foreground/background placeholders: replace files under `assets/icons`
- Feature graphic: 1024x500 PNG
- Phone screenshots: portrait gameplay, level select, perfect push, level complete
- Category: Game / Arcade or Casual
- Tags: arcade, brick breaker, physics, action, casual
- Content rating notes: arcade physics, no user generated content, no data sharing, no ads in this version
- Privacy policy: use `docs/privacy-policy-placeholder.md` until a hosted policy is created
- Data Safety: no data collected, no data shared, offline localStorage save only

Google Play release reminders:

- Upload an Android App Bundle (`.aab`), not a raw APK.
- Enroll in Play App Signing so Google Play manages and protects the app signing key.
- Target Android 15 / API 35 or higher for new apps and updates after August 31, 2025. Re-check the current Play requirement before final submission.
- Test on a physical Android device before production.
